How to access QuakerCraft

These instructions assume that you have already bought, downloaded, and installed MineCraft: Java Edition and have a screenname that you have given to Finn, and that he has told you that you have been added to the server.

I strongly recommend joining the discord server for further assistance, as Minecraft can be very confusing to start.

First off, find the minecraft launcher. The icon looks like this:

In the launcher, select "Minecraft: Java Edition", then the big green "Play" in the bottom center

When minecraft boots and you see the main menu, select "Minecraft Realms"

QuakerCraft should be right there in the middle. Select it, and then select "Play."
