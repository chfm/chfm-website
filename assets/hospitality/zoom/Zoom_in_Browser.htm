Run Zoom in a Browser

If you aren't able to install Zoom Client for Meetings on you computer, you can still run Zoom in a browser (Chrome, Firefox, Safari, etc).
However, functionality varies with the browser—you might not see what others see (works best in Chrome).

1. Click Click here to open Zoom in your browser.

2. Alternatively, you can click download & run Zoom to first install Zoom on your computer

However, there will be a delay before you can join the meeting, so you might want to run Zoom in your browser and install it later.
